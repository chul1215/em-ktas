# PRD: 응급실 KTAS 검사처방 분석 대시보드

> **문서 버전:** v1.0
> **작성일:** 2026-03-06
> **프로젝트명:** EM-KTAS Dashboard
> **목적:** 대전선병원·유성선병원 응급실 KTAS 등급별 검사처방 현황, KPI 달성도, 수익 분석을 한눈에 파악할 수 있는 대시보드

---

## 1. 프로젝트 개요

### 1.1 배경
- 양 병원 응급실의 KTAS 등급별 기본lab 처방율, 의사별 처방 편차, 수익 구조를 통합 모니터링할 필요성
- 신규 KPI 체계(100%/100%/93%/75%/20%) 적용에 따른 목표 대비 현황 실시간 추적
- ROOMi(적정진료관리팀 AI)의 분석 견해를 현장에 전달하기 위한 인터페이스

### 1.2 사용자
- 응급실 운영 관리자
- 적정진료관리팀
- 병원 경영진

### 1.3 기술 스택
- **프론트엔드:** HTML + CSS + JavaScript (단일 파일 또는 소수 파일)
- **차트 라이브러리:** Chart.js 또는 ApexCharts (CDN)
- **데이터:** JSON 내장 (정적 데이터 기반, 별도 백엔드 없음)
- **배포:** 로컬 HTML 파일로 브라우저에서 바로 실행

### 4.3 [B] KTAS 등급별 현황 테이블 + KPI 게이지

**좌측 카드 (6컬럼):**

테이블 구성:

| KTAS | 내원건수 | 현재 처방율 | KPI 목표 | 달성상태 | 입원율 | 환자당 평균금액 |
|------|---------|-----------|---------|---------|-------|---------------|

- 각 KTAS 행마다 **처방율 게이지 바** 표시 (현재 처방율 / KPI 목표를 시각적으로 비교)
- 달성상태: KPI 초과 시 초록 체크, 미달 시 빨강 X
- KTAS 5등급은 현재 처방율이 KPI보다 과도하게 높으면 **빨강 경고** (과잉처방 표시)

**색상 규칙:**
- 현재 ≥ KPI: 초록 (`#27AE60`)
- 현재 < KPI: 빨강 (`#E74C3C`)
- 현재 > KPI × 1.5 (KTAS 4-5): 오렌지 경고 (`#F39C12`) — 과잉처방 의심

### 4.5 [D] 의료진별 처방 현황 (핵심 섹션)

**전체 너비 카드 (12컬럼):**

**상단: 뷰 전환 탭**
- `[처방율 뷰]` | `[평균금액 뷰]` | `[KPI 비교 뷰]`

#### 4.5.1 처방율 뷰 (기본)

테이블 + 히트맵 스타일:

| 의사명 | KTAS 1(%) | KTAS 2(%) | KTAS 3(%) | KTAS 4(%) | KTAS 5(%) | 전체(%) |
|--------|-----------|-----------|-----------|-----------|-----------|---------|

- 각 셀은 **히트맵 색상** 적용:
  - KPI 이상: 초록 배경
  - KPI 미만: 빨강 배경
  - KPI 대비 과도 초과 (KTAS 4-5에서 KPI×1.5 이상): 오렌지 배경
- 행 호버 시 해당 의사의 전체 통계 하이라이트

#### 4.5.2 평균금액 뷰

| 의사명 | 환자수 | 환자당 평균금액 | 환자당 중앙값 | 평균 처방건수 | 기본lab 평균처방수 |
|--------|-------|---------------|-------------|-------------|-----------------|

- 평균금액에 바 게이지 표시 (전체 평균 대비)
- 최고/최저 의사 하이라이트

#### 4.5.3 KPI 비교 뷰

| 의사명 | 현재 전체 처방율 | KPI 목표 환산 | 현재 수익 기여 | KPI 적용 시 수익 | 차액 |
|--------|----------------|-------------|--------------|----------------|------|

- 각 의사별 KPI 적용 전/후 수익 변화를 시각화
- **차트:** 의사별 수직 바 차트 (현재 수익 vs KPI 적용 수익)

### 4.9 [H] ROOMi 분석 견해

**좌측 카드 (7컬럼):**

#### 실행 로드맵 탭

## 5. 인터랙션 명세

### 5.1 병원 전환 스위치
- 토글 클릭 시 전체 대시보드 데이터가 해당 병원으로 전환
- 전환 시 페이드 애니메이션 (300ms)
- 현재 선택 병원: 네이비 배경 + 흰색 텍스트
- 비선택 병원: 투명 배경 + 라이트 텍스트

### 5.2 차트 인터랙션
- 호버: 툴팁으로 상세 수치 표시
- 클릭: 해당 KTAS 등급 또는 의사의 상세 정보 하이라이트

### 5.3 테이블 인터랙션
- 컬럼 헤더 클릭: 정렬 토글 (오름차순/내림차순)
- 행 호버: 하이라이트 배경색
- 의사명 클릭: 해당 의사의 KTAS 분포 상세 팝오버

### 5.4 탭 전환
- 각 카드 내 탭 클릭 시 콘텐츠 전환
- 언더라인 애니메이션으로 활성 탭 표시
